# AI / Machine Learning Study Project

## 1. Overview
This project focuses on understanding fundamental machine learning and deep learning algorithms.  
Various models were implemented and compared to analyze their characteristics and performance.

---

## 2. Project Structure

### 1) Regression
- Linear Regression (Simple / Multiple)
- Polynomial Regression  
→ Predict numerical values such as salary and company profit

### 2) Classification
- Logistic Regression
- K-Nearest Neighbors (KNN)  
→ Predict categorical outcomes such as user behavior and class labels

### 3) Deep Learning
- Artificial Neural Network (ANN)
- Convolutional Neural Network (CNN)
- Transfer Learning  
→ Image classification and neural network structure understanding
